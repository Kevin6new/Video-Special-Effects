Video-special effects 

This project focuses on understanding and applying the fundamentals of image and video processing using the C++ programming language and OpenCV. Some extensions are created using HTML/CSS, Javascript and Python.

Google Drive link for all videos of the assignment is in the following link: https://drive.google.com/drive/folders/12jw3SlCWOz768X9te65ukVmmEDm7csVU?usp=drive_link
